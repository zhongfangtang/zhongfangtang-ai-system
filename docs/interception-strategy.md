## 二、六大平台截流场景定位

### 2.1 抖音截流场景

```
场景1：评论区截流（主战场）
├─ 目标：自己视频评论区 + 同城热门视频评论区 + 养生/护肤垂类视频评论区
├─ 方法：AI监控评论→关键词匹配→自动回复引导
├─ 优势：用户意图明确，转化路径短
└─ 频率：每15分钟扫描，日均≤100条互动

场景2：私信截流
├─ 目标：主动咨询的用户 + 评论区引导来的用户
├─ 方法：AI自动回复→体质测评引导→小程序链接
└─ 频率：实时响应

场景3：同城截流
├─ 目标：宜昌同城发布内容的用户
├─ 方法：监控同城标签+宜昌POI→关注+互动→建立连接
└─ 频率：每日扫描，精选20条互动

场景4：搜索截流
├─ 目标：搜索"体质调理""精油""SPA宜昌"等关键词的用户
├─ 方法：SEO优化标题+话题标签+评论区置顶引导
└─ 特点：被动截流，搜索来的用户意向最高
```

### 2.2 小红书截流场景

```
场景1：笔记评论区截流
├─ 目标：自己笔记评论区 + 养生/护肤/美体垂类笔记评论区
├─ 方法：AI监控→生成回复话术→人工确认发送
├─ 注意：小红书风控严格，不自动发送
└─ 频率：每30分钟扫描，日均≤50条互动

场景2：私信截流
├─ 目标：评论引导来私信的用户
├─ 方法：AI生成话术→人工发送→引导小程序
├─ 注意：私信含联系方式高风险
└─ 策略：先建立信任，再引导主页

场景3：搜索截流
├─ 目标：搜索"精油推荐""敏感肌护肤""中医养生"的用户
├─ 方法：笔记SEO优化（标题+标签+正文关键词）
├─ 策略：搜索页是第一入口，长尾关键词覆盖
└─ 特点：小红书搜索流量占比>40%

场景4：话题截流
├─ 目标：参与热门话题讨论的用户
├─ 方法：发布话题相关笔记→评论区互动→关注目标用户
```

```
  ├─ 7天内跟进

20-39分 → C级（低意向）
  ├─ 简单互动或点赞
  ├─ 不主动推送CRM
  ├─ 加入观察列表
  └─ 7天后复查互动情况

0-19分 → D级（无意向）
  ├─ 不互动
  ├─ 不记录
  └─ 仅统计曝光
```

## 七、截流效果评估指标

### 7.1 核心指标体系

#### 过程指标（每日监控）
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|---------|--------|---------|
| 扫描量 | 各平台评论区扫描总数 | 1000+/日 | <500 |
| 命中量 | 关键词命中的评论数 | 100+/日 | <50 |
| 互动量 | 实际发出互动数 | 50+/日 | <20 |
| 互动率 | 互动量÷命中量 | >50% | <30% |
| 屏蔽率 | 屏蔽词命中÷扫描量 | <5% | >10% |

#### 转化指标（每周监控）
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|---------|--------|---------|
| 有效线索数 | 意向评分≥60的用户数 | 30+/周 | <15 |
| 线索→小程序转化率 | 进入小程序用户÷线索数 | >15% | <10% |
| 线索→到店转化率 | 到店用户÷线索数 | >5% | <3% |
| 线索→消费转化率 | 消费用户÷线索数 | >3% | <1% |
| 单线索成本 | 总投入÷有效线索数 | <30元 | >50元 |

#### 效率指标（每月评估）

|------|---------|--------|
| 渠道ROI | 渠道收入÷渠道投入 | >3:1 |
| 平台效率排名 | 线索量×转化率排名 | 动态调整 |
| 关键词效率排名 | 关键词线索量×转化率 | 动态优化 |
| 话术效率排名 | 话术线索量×转化率 | A/B测试 |

```

   - 调整权重参数
```

## 八、合规与风险控制

### 8.1 互动频率上限

| 平台 | 每小时上限 | 每日上限 | 互动间隔 | 说明 |
|------|-----------|---------|---------|------|
| 抖音 | 20条 | 100条 | 2-5分钟随机 | 企业号上限更高 |
| 小红书 | 10条 | 50条 | 5-10分钟随机 | 风控极严，宁慢勿快 |
| 快手 | 15条 | 80条 | 2-5分钟随机 | 企业号 |
| B站 | 5条 | 30条 | 10-15分钟随机 | 质量优先 |
| 百家号 | 5条 | 20条 | 10-15分钟随机 | 评论量少 |
| 视频号 | 3条 | 20条 | 手动 | 无API |

### 8.2 禁止操作清单

```
❌ 在同一视频下集中回复多条评论
❌ 使用完全相同的话术回复不同用户
❌ 在评论区直接留联系方式
❌ 在竞品内容下截流
❌ 使用医疗功效词
❌ 承诺具体效果（如"7天见效"）
❌ 贬低或攻击竞品
❌ 发布虚假客户案例
❌ 在敏感话题（政治/医疗）下互动
❌ 使用自动化工具绕过平台限制
```

### 8.3 应急处理流程

```
1. 收到平台警告
   → 立即暂停该平台所有自动互动
   → 审查近期互动内容
   → 调整话术和频率
   → 观察48小时后恢复

2. 账号限流/降权
   → 暂停自动互动7天
   → 改为纯人工互动
   → 发布高质量原创内容
   → 观察数据恢复后再逐步自动化

3. 用户投诉
   → 立即停止与该用户互动
   → 如已加私域，发送道歉消息
   → 分析投诉原因
   → 更新屏蔽规则

4. 恶意举报
   → 保存所有互动记录
   → 准备申诉材料
   → 调整策略避开恶意账号
   → 法律途径（必要时）
```
