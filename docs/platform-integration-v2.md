## 二、抖音企业号对接方案

### 2.1 API权限申请流程

```
步骤1：抖音开放平台注册
  ├─ 访问 open.douyin.com
  ├─ 使用已有抖音企业号登录
  └─ 完成企业认证（免费）

步骤2：创建应用
  ├─ 应用类型：企业号应用
  ├─ 应用名称：中芳堂全域AI
  ├─ 应用描述：中医芳香疗法内容创作与客户服务
  └─ 获取 AppID + AppSecret

步骤3：申请API权限
  ├─ 视频管理权限（video.create/video.list/video.data）
  ├─ 评论管理权限（comment.list/comment.reply）
  ├─ 私信管理权限（im.message.send/im.message.list）
  ├─ 用户数据权限（user.info/fans.list/fans.data）
  ├─ 数据统计权限（data.external.item/data.external.user）
  └─ 小程序挂载权限（需单独申请）

步骤4：配置回调地址
  └─ OAuth2回调：https://your-cloudbase.com/douyin/callback
```

### 2.3 风控规避策略

```
1. 发布频率控制
   - 每日发布上限：3条
   - 发布间隔：≥2小时
   - 高峰时段避让：12:00-13:00, 18:00-20:00

2. 互动频率控制
   - 每小时评论回复：≤20条
   - 每日评论回复：≤100条
   - 互动间隔随机：2-5分钟

3. 内容重复检测
   - 同一素材不重复发布
   - 文案查重率<30%
   - MD5校验避免重复上传

4. 账号异常监控
   - 限流预警：播放量<正常值50%
   - 降权预警：搜索不到账号/视频
   - 违规预警：收到系统通知
```

  ├─ 评论管理（查看/回复）- 需单独申请

```

   - 每天登录浏览≥30分钟
   - 点赞收藏其他笔记≥5篇
   - 参与话题讨论
   - 保持真实用户行为模式

4. 防限流策略
   - 新号前7天不发商业内容
   - 内容类型多样化
   - 避免短时间内集中操作
```

## 九、统一风控与合规体系

### 9.1 全域风控规则

```
1. 内容安全红线
   ❌ 医疗功效承诺（治疗/治愈/根治/保证/100%）
   ❌ 疾病诊断描述
   ❌ 处方药物推荐
   ❌ 未经验证的治疗方法
   ❌ 恐怖/恶心的医学图片
   ❌ 虚假宣传和夸大效果
   ❌ 贬低竞品的内容
   ❌ 未经授权的客户肖像

2. 合规替代用语
   治疗 → 调理/养护/改善
   治愈 → 舒缓/平衡/修复
   诊断 → 辨识/评估/分析
   处方 → 方案/搭配/建议
   疗效 → 效果/反馈/体验
   疗程 → 周期/阶段/过程

3. 统一合规声明模板
   "本公司业务属于美容美体养生服务，不涉及医疗服务。
   本内容由AI生成，仅供参考。个体差异存在，效果因人而异。
   如有健康问题，请咨询专业医师。"

4. 各平台差异化合规要点
   - 抖音/快手：商品橱窗需资质，带货需报白
   - 小红书：功效词极其敏感，人设号更安全
   - 视频号：微信生态审核最严，医疗相关一票否决
   - B站：社区规则为主，商业化内容需标注
   - 百家号：百度审核严格，原创度要求高
```

### 9.2 应急响应流程

```
触发条件 → 处置动作

1. 平台违规通知
   → 立即下架问题内容
   → 分析违规原因
   → 更新屏蔽词库
   → 全平台排查同类问题

2. 账号限流/降权
   → 暂停该账号发布
   → 检查近期内容
   → 调整发布策略
   → 72小时观察期

3. 账号封禁
   → 启动备用账号
   → 分析封禁原因
   → 申诉（如可能）
   → 矩阵其他号调整策略

4. 用户投诉
   → 1小时内响应
   → 核实投诉内容
   → 按类型处理（退款/道歉/整改）
   → 记录到知识库
```
